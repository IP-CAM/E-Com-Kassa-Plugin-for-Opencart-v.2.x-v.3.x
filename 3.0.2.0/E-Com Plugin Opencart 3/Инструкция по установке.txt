Установка Модуля:

1. Загрузите файлы из папки Files to upload в корень сайта.
2. Установите файл из папки Plugins to install с помощью адм. раздела Модули->Установка расширений 
3. В разделе Модули->Модификаторы нажмите кнопку обновить
3.1 В Система-Пользователи->Группы пользователей разрешите для administrator просмотр и изменение модуля extension/module/ecomkassa
4. В разделе Модули->Модули активируйте модуль E-COM KASSA
5. Настройте Модуль E-COM KASSA укажите на каких способах оплаты, при установке какого статуса отправлять чек
о продаже. Проверьте подключение к E-COM и измените статус на Включено
6. Настройте CRON для очереди проверки чеков (пример комманды указан в модуле)
7. Чеки отобразятся в просмотре информации о заказе. (Продажи->Заказы->Просмотр)
